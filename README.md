# CPU Monitor Desktoop App

>### A Simple Desktop App With electron.Js and bare minimum project structure `to get started developing with electron`.

#### App Url: https://www.mediafire.com/file/9ixtgv310vj1mv3/Cpu_Monitor-1.0.0_Setup.exe/file

## Dependencies:star:
| dependencies | Version |
| --- | ----------- |
| npm | 8.3.1 |
| Electron | 17.1.0 |
| node.js | 16.14.0 |

### We’ll install Electron Forge using npm
```
npm install create-electron-app -g
```
### After the install has finished, create a new project by typing create-electron-app follow by a project name. Let’s name it cool-app.
```
create-electron-app cool-app
```
### Run App
```
npm run start
```
### Creating an Installer
```
npm run make
```
